Fonts and Text

attribute example values details
font-size 2em, 22px font size, either in em or px
font-weight bold, 100 choose how bold the font will be
font-style italic, bold The style you want to apply to the text
font-family Helvetica, sans-serif, Lato Chooses the font
text-align left,center, justify where the text should align itself to the parent element
list-style square, circle Select how the list should display as

Borders and Placement

attribute example values details
margin auto, 20px create space around elements, outside of any defined borders. if auto is selected, will center the element
width 60% width to take relative to parent element
height auto, 300px, 30% Choose the height of the element inside the padding, border and margin of the element
padding 20px,5% Use to create space around an element's content, inside of the defined borders
border 1px solid red defines how the border of the element will look like. border-style must be set for it to display
border-style solid,none, hidden, dashed, double, groove defines what type the border should be
line-height 24px, 20%, 3 sets the height of a line
float left, right specifies how an element should float
opacity 0, 0.1, 0.9, 1 Select opacity of element. Goes from 0 to 1
position relative, static, fixed, absolute, sticky specifies type of positioning method used for an element
box-shadow 5px 0px 5px 0px white sets a shadow to an element

Transformations

attribute example values details
text-transform uppercase, lowercase, capitalize changes the case of the text
!important color:blue !important Will override existing tag attribute. Very hard to override this

Background

attribute example values details
display block, none, inline selects how the element should display as
background-color white, red, #FFF Selects the color of the background for the element
background rgba(0,128,0,0.3) Shorthand property for background-color and opacity
background-image url(http://...) places a background image. Either external or internal links
background-size auto, cover, %, contain
background-position center center, x% y%, xpos ypos sets the starting position of the background image
